# gemini-ai-chat

## Setup

1. Clona el repo
   git clone https://github.com/ing-thejis/gemini-ai-chat.git

2. Configura el backend
   cd backend
   cp .env.example .env   # agrega tu GEMINI_API_KEY
   pnpm install
   pnpm run dev

3. Configura el frontend
   cd frontend
   cp .env.example .env
   pnpm install
   pnpm run dev
